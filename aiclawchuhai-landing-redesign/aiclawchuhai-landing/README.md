# AIClaw 出海增长落地页

这是一个适合部署到 Cloudflare Pages 的纯静态落地页项目。

## 已包含

- 更现代的视觉重构
- 滚动 reveal 动画
- 浮动卡片与图表动画
- Telegram / WhatsApp / 微信悬浮联系按钮
- 社媒 logo 组件
- SEO / GEO 基础配置

## 部署

1. 新建 GitHub 仓库
2. 上传本目录全部文件到仓库根目录
3. 打开 Cloudflare Pages，选择 Connect to Git
4. 连接仓库
5. Build command 留空
6. Build output directory 填 `/`
7. 绑定 `aiclawchuhai.shop`

## 可自行替换

- `logo.png`
- `og-cover.jpg`
- 页面文案
- 联系方式
